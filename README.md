# Bulmacomp
[https://viewcomponent.org/](ViewComponent) collection for [https://bulma.io/](bulma) css framework

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "bulmacomp"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install bulmacomp
```
